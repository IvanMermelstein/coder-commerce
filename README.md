## Proyecto de E-Commerce: Pi-Pizza

### Que es?
Este es el proyecto final para el curso de React.js de CoderHouse. Es un e-commerce dedicado a la mejor comida del mundo: la pizza.

### Instalacion 
Como lo instalo? En el directorio del proyecto:
```
npm install
```
### Ejecucion
Como lo ejecuto? En el directorio del proyecto:
```
npm start
```
Para verlo podes abrir en tu navegador la siguiente ruta [http://localhost:3000](http://localhost:3000)
